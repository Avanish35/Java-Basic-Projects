# 🎮 Number Guessing Game (Java)

A simple and interactive **Number Guessing Game** built using Java.
The program generates a random number, and the player tries to guess it with helpful hints like **Hot 🔥**, **Cold ❄️**, or **Right 🎯**.

---

## 🚀 Features

* 🎲 Random number generation (1–100)
* 🔥 Hint system (Hot / Cold / Right)
* 🔁 Continuous guessing until correct
* 🧠 Logic-based feedback using range difference
* 💻 Console-based gameplay

---

## 🛠️ Tech Stack

* **Language:** Java
* **Concepts Used:**

  * Loops (`do-while`)
  * Conditional Statements (`if-else`)
  * Functions/Methods
  * Random Number Generation
  * User Input (`Scanner`)

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/number-guessing-game.git
```

2. Navigate to the project folder:

```
cd number-guessing-game
```

3. Compile the Java file:

```
javac NumberGuessingGame.java
```

4. Run the program:

```
java NumberGuessingGame
```

---

## 🎯 How to Play

* Enter a number between **1 and 100**

* The program will give hints:

  * 🔥 **Hot** → Close to the correct number
  * ❄️ **Cold** → Far from the number
  * 🎯 **Right** → Correct guess

* Keep guessing until you win!

---

## 💡 Example

```
Enter a number between 1 and 100: 50
Hint: Cold

Enter a number between 1 and 100: 72

Enter a number between 1 and 100: 68
🎉 You guessed it Right!
```
