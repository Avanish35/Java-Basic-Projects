# 🧮 Java Console Calculator

A simple console-based calculator built using Java.
It allows users to perform basic arithmetic operations through a menu-driven interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division (with zero handling)
* 🔁 Continuous execution using loop
* ❌ Exit option

---

## 🛠️ Technologies Used

* Java
* Scanner (for user input)

---

## 📂 Project Structure

```
Calculator.java
README.md
```

---

## ▶️ How to Run

1. Compile the program:

```
javac Calculator.java
```

2. Run the program:

```
java Calculator
```

---

## 💡 How It Works

* The program displays a menu with operations.
* User selects an option.
* User enters two numbers.
* The selected operation is performed.
* The result is displayed.
* The program repeats until the user chooses to exit.

---

## ⚠️ Edge Case Handling

* Division by zero is handled with an error message.
* Invalid menu choices are handled gracefully.
